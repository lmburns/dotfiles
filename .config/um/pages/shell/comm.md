## SYNOPSIS
compare two sorted files line by line

Returns 3 columns
: lines unique to first file
: lines unique to second file
: lines in common

## OPTIONS

`-1`, `-2`, `-3`
: supress one of the mentioned lines

`-12`
: get only common words

`--nocheck-order`
: different sort order
