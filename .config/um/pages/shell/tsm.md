## SYNOPSIS
Transmission-remote / daemon

## OPTIONS

Watch downloads
: `watch -n 0.1 'transmission-remote -l'`

Dump settings
: `tsmd --dump-settings`

Add
: `tsm -a <file>`

Remove
: `tsm -t 3,4 -r`

Remove all
: `tsm -t all -r`
