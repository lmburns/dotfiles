## OPTIONS

`whence -m`
: matching a pattern

`whence -v`
: is equivalent to type

`whence -p`
: is equivalent to path

`whence -c`
: is equivalent to which

`whence -c`
: is equivalent to where

`whence`
: is equivalent to command -v
