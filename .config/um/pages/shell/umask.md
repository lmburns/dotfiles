# umask --
{:data-section="shell"}
{:data-date="March 26, 2021"}
{:data-extra="Um Pages"}

## SYNOPSIS
Change permissions

## OPTIONS


|-------------|-------------------------|
| Octal value | Permission              |
|-------------|-------------------------|
| 1.          | read, write and execute |
|-------------|-------------------------|
| 2.          | read and write          |
|-------------|-------------------------|
| 3.          | read and execute        |
|-------------|-------------------------|
| 4.          | read only               |
|-------------|-------------------------|
| 5.          | write and execute       |
|-------------|-------------------------|
| 6.          | write only              |
|-------------|-------------------------|
| 7.          | execute only            |
|-------------|-------------------------|
| 8.          | no permissions          |
|-------------|-------------------------|

|--------|------------------|-----------------|
| Unmask | File Permissions | Directory Perms |
|--------|------------------|-----------------|
| 002    | -rw-rw-r--       | drwxrwxr-x      |
|--------|------------------|-----------------|
| 007    | -rw-rw----       | drwxrwx---      |
|--------|------------------|-----------------|
| 022    | -rw-r--r--       | drwxr-xr-x      |
|--------|------------------|-----------------|
| 027    | -rw-r-----       | drwxr-x---      |
|--------|------------------|-----------------|
| 077    | -rw-------       | drwx------      |
|--------|------------------|-----------------|
