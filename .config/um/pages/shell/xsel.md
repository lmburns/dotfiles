# xsel --
{:data-section="shell"}
{:data-date="June 10, 2021"}
{:data-extra="Um Pages"}

## SYNOPSIS
X-Clipboard

## OPTIONS

`xsel < file`
: copy file

`xsel > file`
: write file

`xsel --append < file`
: append file to X

`xsel --follow < file`
: follow a growing file

`xsel --delete`
: Will cause the program in which text is selected to delete that text

`xsel --secondary < file`
: secondary exchange

`xsel --secondary > file`
: secondary exchange

`xsel --exchange`
: swap with primary

`xsel --clipboard < file`
: standard clipboard

`xsel --keep`
: keep past closing of program
